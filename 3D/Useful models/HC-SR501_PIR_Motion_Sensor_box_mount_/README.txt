# Summary

Крепление на стену. Данная коробка только для самого датчика HC-SR501. Вся обвязка и реле монтируются отдельно. Датчик можно вмонтировать в галтель и его почти не будет видно.  *Всем Добра!*

-----------------------------------
Wall mount. This box is only for the sensor HC-SR501. All piping and relays are mounted separately. The sensor can be built into the fillet and it is almost not visible. All Good!
